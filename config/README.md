# Config Klasörü

Bu klasör, uygulama yapılandırma dosyalarınız için kullanılabilir.

Örnek kullanım:
- `config.json` - Genel uygulama ayarları
- `credentials.json` - API anahtarları (dikkat: `.gitignore`'a ekleyin!)
- `settings.json` - Kullanıcı tercihleri

**Not:** Hassas bilgiler içeren dosyaları `.gitignore`'a eklemeyi unutmayın!
