Yazılım Projesi – Replit + GitHub + .exe

Amaç:

Replit üzerinde Python ile bir masaüstü uygulaması (Windows) geliştirilmesi

Proje Git ile GitHub’a bağlanacak.

GitHub Actions ile her push sonrası otomatik olarak .exe build edilecek ve “Artifacts”/“Release” olarak indirilebilir olacak.

Scrape fonksiyonlarını ve gerekli diğer dosyaları daha sonra ben ekleyeceğim, o yüzden altyapıyı ve butonları sen hazırla.

İstenenler:

Teknoloji

Dil: Python 3.x

UI: Tercihen tkinter (ekstra bağımlılık gerekmesin) veya PyQt5 (daha şık, ama bunda karar verebilirsin)

.exe üretimi: PyInstaller ile tek dosya .exe

Proje yapısı (Replit içinde)

Klasör yapısı örnek:

main.py → Uygulamanın giriş noktası, UI ve butonlar burada

scraper/ → (boş bırak, içine scrapeleri ben koyacağım)

config/ (opsiyonel) → config.json vs. koymak için

requirements.txt → Kullanılan tüm Python paketleri

Kod scraper klasöründeki fonksiyonları çağırabilecek şekilde tasarlanacak (örn: from scraper.core import run_scraper gibi, ben o fonksiyonu sonradan eklerim).

Arayüz (şimdilik basit, ama net):

Pencere başlığı: “SmartXFlow – Odds & Volume Monitor” (isim örnek)

Ana bileşenler:

“Şimdi Scrape Et” butonu

Şimdilik print("Scrape başladı...") yapsın, ben sonra gerçek fonksiyonu eklerim.

“Otomatik Scrape Başlat” butonu

Zaman aralığını Entry veya Dropdown ile kullanıcıdan alsın (ör.: 1, 5, 10 dakika)

Arka planda belirlenen aralıkla run_scraper() çağıracak şekilde zamanlayıcı mantığını hazırlayın (thread/timer vs., temiz implementasyon).

“Durdur” butonu

Otomatik scrape döngüsünü temiz bir şekilde durdursun.

Alt kısımda:

Basit bir log penceresi (Text alanı) olsun. Her işlemi buraya yazsın (örn: “10:32 – Scrape bitti, X kayıt çekildi”).

Consolesuz .exe (mümkünse)

PyInstaller kullanırken --noconsole parametresi ile arka planda konsol penceresi açılmasın.

GitHub Entegrasyonu

Replit’te proje oluşturulacak ve Git ile GitHub repo’ya bağlanacak.

main branch’e push sonrası:

GitHub Actions workflow’u çalışacak.

Windows runner üzerinde:

pip install -r requirements.txt

pip install pyinstaller

pyinstaller --onefile --noconsole main.py veya giriş dosyası neyse o

dist/ klasöründeki .exe dosyası Actions “Artifacts” kısmına yüklenecek.

Teslim

Replit projesi tam çalışır halde olacak.

GitHub repo bağlantısı yapılmış olacak.

GitHub Actions workflow dosyası (.github/workflows/build.yml) hazır olacak.

Ben repo’ya push/pull yapabileyim, sadece scraping kodlarımı scraper/ klasörüne ekleyip kullanabileyim.

Notlar

Scraper kodlarının iç mantığına şu an gerek yok, sadece:

scraper klasörü

Örnek bir core.py içinde dummy fonksiyon (örn: def run_scraper(): print("dummy")) bırakın, ben üstüne yazacağım.

Kod sade, okunabilir ve yorum satırlarıyla anlaşılır olsun.
